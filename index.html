Repository

Truy cập trang repo trên GitHub để bật GitHub Pages (chọn branch và folder). Nếu muốn, bạn có thể dùng branch gh-pages hoặc bật Pages từ branch main.

Mở repo

Hướng dẫn bật GitHub Pages

  1. Đẩy (commit & push) `index.html` lên repository trên GitHub.
  2. Vào Settings → Pages, chọn branch (ví dụ: main) và thư mục / (root).
  3. Lưu thiết lập; chờ vài phút để trang được build. URL sẽ hiển thị trong phần Pages.

Phiên bản nhỏ gọn — sửa nội dung này trong index.html nếu cần. Nếu bạn muốn, tôi có thể thêm favicon, OG tags, hoặc tự động hóa build.
